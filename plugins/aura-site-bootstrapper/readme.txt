== Description ==

Aura Site Bootstrapper is a powerful plugin designed to accelerate WordPress site setup by automatically creating essential pages, configuring menus, and applying best practices for new websites.

**Key Features:**

* **Essential Pages Creation**: Automatically generate common pages like About Us, Services, Contact, Privacy Policy, Terms of Service
* **Menu Configuration**: Setup primary navigation menus with proper structure
* **SEO-Ready**: Pages come with basic SEO optimization
* **Theme Compatible**: Works seamlessly with any WordPress theme, especially Aura themes
* **One-Click Setup**: Bootstrap entire site structure with minimal effort

**Perfect for:**
- Web developers and agencies
- WordPress site builders
- Businesses launching new websites
- Anyone wanting to speed up WordPress setup

**Created by Aura Marketing** - Leading digital agency specializing in WordPress solutions.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/aura-site-bootstrapper/` or install through WordPress admin
2. Activate the plugin through the 'Plugins' screen
3. Navigate to Tools â†’ Site Bootstrapper to start configuring your site
4. Use the one-click options to create pages and setup menus

== Frequently Asked Questions ==

= Does this work with any theme? =
Yes! While optimized for Aura themes, it works with any WordPress theme.

= Will this overwrite existing content? =
No, the plugin checks for existing pages and menus before creating new ones.

= Can I customize the generated pages? =
Absolutely! All generated content can be edited through the standard WordPress editor.

= Is this plugin free? =
Yes, Aura Site Bootstrapper is completely free and open source.
